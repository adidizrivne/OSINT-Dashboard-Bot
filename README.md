# 🤖 OSINT Dashboard Bot для Telegram

Повнофункціональний бот для **OSINT дослідження** в Telegram. Інтегрує кілька потужних інструментів для пошуку інформації. Працює як на **локальному ПК** так і на серверах.

---

## 📋 Функціональність

### 1. **🔍 Sherlock - Username Пошук**
- Шукає username на **300+ сайтах** (Reddit, Twitter, GitHub, Instagram тощо)
- Швидка ідентифікація присутності користувача в інтернеті
- Повертає прямі посилання на профілі

### 2. **🌍 GeoIP - Геолокація**
- Визначає географічне розташування за IP адресою
- Повертає: країна, регіон, місто, координати
- Інформація про ISP та організацію
- Визначає VPN/Proxy використання
- Використовує IP-API (безплатний, точність ±10км)

### 3. **🏢 WHOIS - Інформація про домени**
- Дані про реєстрацію домену
- Реєстратор, дати створення/оновлення/закінчення
- Nameservers та контактна інформація
- Корисно для дослідження сайтів

### 4. **🔓 HaveIBeenPwned - Перевірка витоків**
- Перевіряє, чи був email скомпрометований
- Показує список баз даних де знайдено email
- Важно для безпеки та приватності
- Дані від Трояна Ханта (офіційна база)

### 5. **📧 Email Search - Пошук в базах**
- Аналізує репутацію email адреси
- Перевіряє доставляємість
- Визначає формат та валідність
- Зв'язує email з відомими витоками

## 🚀 Встановлення - 2 способи

### Способ 1: Автоматичне налаштування (РЕКОМЕНДОВАНО) ⭐

**Найпростіший спосіб!**

```bash

# 2. Введи Bot Token від @BotFather
# 3. Опціонально введи API ключи (можна пропустити Enter)
# 4. Файл .env буде створено автоматично
```

**Setup автоматично:**
- ✅ Перевіримо Python версію
- ✅ Встановить залежності з requirements.txt
- ✅ Запросить Bot Token
- ✅ Попросить опціональні API ключі
- ✅ Створить .env файл
- ✅ Протестує підключення

**Час:** ~5-10 хвилин

### Способ 2: Ручна установка

## 🛠️ Вирішення Проблем

### "python: command not found"
```bash
# Windows: Переустановити Python (додати до PATH)
# Mac: brew install python3
# Linux: sudo apt install python3
```

### "ModuleNotFoundError"
```bash
pip install -r requirements.txt
```

### "Bot Token не знайдений"
```bash
# Перевір .env файл:
# TELEGRAM_BOT_TOKEN=123456789:ABCdef...
```

### "Sherlock не знайшов результати"
- Sherlock потребує 2-5 хвилин ⏳
- Спробуй з популярним username
- Перевір інтернет з'єднання

- Перевір інтернет з'єднання
- Спробуй пізніше (тимчасовий збій)
- Використай VPN якщо заблоковані сайти
