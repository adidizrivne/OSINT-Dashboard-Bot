# 🤖 OSINT Dashboard Bot для Telegram

Повнофункціональний бот для **OSINT дослідження** в Telegram. Інтегрує кілька потужних інструментів для пошуку інформації.

---

## 📋 Функціональність

### 1. **🔍 Sherlock - Username Пошук**
- Шукає username на **300+ сайтах** (Reddit, Twitter, GitHub, Instagram тощо)
- Швидка ідентифікація присутності користувача в інтернеті
- Повертає прямі посилання на профілі

### 2. **🌍 GeoIP - Геолокація**
- Визначає географічне розташування за IP адресою
- Повертає: країна, регіон, місто, координати
- Інформація про ISP та організацію
- Використовує IP-API (безплатний, точність ±10км)

### 3. **🏢 WHOIS - Інформація про домени**
- Дані про реєстрацію домену
- Реєстратор, дати створення/оновлення/закінчення
- Nameservers та контактна інформація
- Корисно для дослідження сайтів

### 4. **🔓 HaveIBeenPwned - Перевірка витоків**
- Перевіряє, чи був email скомпрометований
- Показує список баз даних де знайдено email
- Важно для безпеки та приватності
- Дані від Трояна Ханта (офіційна база)

### 5. **📧 Email Search - Пошук в базах**
- Аналізує репутацію email адреси
- Перевіряє доставляємість
- Визначає формат та валідність
- Зв'язує email з відомими витоками

## ⚙️ Встановлення

### Крок 1: Отримання Telegram Bot Token

1. Відкрий Telegram і знайди бота **@BotFather**
2. Напиши `/start` 
3. Напиши `/newbot`
4. Дай назву своєму боту (напр. "My OSINT Bot")
5. Дай username (має закінчуватися на `_bot`, напр. "my_osint_bot")
6. Скопіюй отриманий **token**

### Крок 2: Встановлення залежностей

```bash
# Клонуй репозиторій (або завантаж файли)
git clone <url>
cd osint-bot

# Установи Python залежності
pip install -r requirements.txt

# Встановлення додаткових компонентів
pip install --break-system-packages sherlock-project
```

### Крок 3: Налаштування змінних оточення

**Варіант A (Lokальний запуск):**
```bash
export TELEGRAM_BOT_TOKEN="your_token_here"
python3 osint_bot.py
```

**Варіант B (Введення при запуску):**
```bash
python3 osint_bot.py
# Введи токен коли запросить
```

**Варіант C (Через .env файл):**
```bash
# Створи файл .env
echo "TELEGRAM_BOT_TOKEN=your_token_here" > .env

# Встановлення python-dotenv
pip install python-dotenv

# Запуск з автозавантаженням
python3 osint_bot.py
```

```
│   ├── whois_lookup() - інформація про домени

    ├── start() - стартова команда

```

## 🔑 API & Джерела Даних

### Безплатні API (без ключа):
- **IP-API** - GeoIP (безплатно до 45 запитів/хвилину)
- **WHOIS** - python-whois (безплатно)
- **Sherlock** - 300+ сайтів (безплатно, 2-5 хвилин на запит)
- **HaveIBeenPwned** - безплатно без ключа (деякі обмеження)
- **EmailRep** - безплатно з публічним ключем

### Ліміти та ограничення:
```
IP-API: 45 запитів/хвилину без ключа
WHOIS: Без офіційних ліміту
Sherlock: ~2-5 хв на один username
HIBP: 1500 запитів/год без ключа
EmailRep: 100 запитів/день без ключа
```

## ⚙️ Розширена Конфігурація

### Додавання свого API ключа (опціонально)

Відредагуй `osint_config.json` після першого запуску:

```json
{
  "api_keys": {
    "ipqualityscore": "твій_ключ",
    "abuseipdb": "твій_ключ",
    "hibp": "твій_ключ"
  }
}
```

**Де отримати ключі:**
- **IPQualityScore** - https://www.ipqualityscore.com/ (2000 запитів/місяць)
- **AbuseIPDB** - https://www.abuseipdb.com/ (3000 запитів/день)
- **HIBP** - https://haveibeenpwned.com/API/v3 (1.5 млн запитів/год)

## 🐛 Вирішення проблем

### Помилка: "Module not found"
```bash
pip install -r requirements.txt --break-system-packages
```

### Sherlock займає занадто довго
- Це нормально, пошук на 300+ сайтах займає 2-5 хвилин
- Бот автоматично покаже таймаут повідомлення

### IP-API не працює
- Перевір своє інтернет з'єднання
- API може бути недоступний в деяких країнах (використай VPN)

### Telegram бот не відповідає
- Переконайся, що токен правильний
- Перезагрузи бота: `Ctrl+C` и запусти знову
